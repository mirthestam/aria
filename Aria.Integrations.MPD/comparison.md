# MPD Clients
Comparison of features with other MPD clients,that are currently being maintained.
I want to use this list to compare features and for inspiration.

I select these based upon feature completeness and whether they are maintained.

Last updated: 2025-12-27

 - ncmmpcpp

Web

 - [AMpd](https://github.com/rain0r/ampd) Mobile friendly.
